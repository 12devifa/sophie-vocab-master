# 🎧 SOPHIE: Vocab Master
> **"Your hands-free learning universe, designed for total focus."**

SOPHIE is a high-performance language immersion tool (French 🇫🇷 - German 🇩🇪) inspired by the clean, card-based interface of **NotebookLM**. It is designed for students of all ages who want to turn commute time into productive learning.

## ✨ Key Features
* **🧼 Default Clean-up:** Every session starts with a blank canvas. No residual data from previous lessons to ensure 100% current focus.
* **🎙️ Hands-Free Mode (Echo Rhythm):** Automated sequential playback: 
    * `Listen (FR) -> Mental Pause (Silence) -> Confirmation (DE)`.
* **🖼️ Notebook Gallery:** Visual organization using smart cards with automatic icons and pastel themes.
* **💾 Privacy & Performance:** No heavy databases. Local storage and `.json` export/import to keep the web fast, light, and private.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript.
* **Audio Engine:** Web Speech API (Native browser neural voices).
* **Architecture:** Single Page Application (SPA).

## 🚀 How to use
1. Paste your text or upload a vocabulary image.
2. Process the lesson to extract key terms.
3. Hit **Play** and let SOPHIE guide your ears and mind.
